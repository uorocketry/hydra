Standard Frame 
--------------
First sync char -> 0xFFu
Second sync char -> 0x5Au
Msg Id
Msg class
Msg size
Msg data
CRC calculated from Id, Class, Size, and Data.
End of frame char -> 0x33u
Example get device info:
ff 5a 04 10 00 00 79 f7 33
Msg Id -> 04
Msg class -> 10
Msg size -> 00 00
Msg data -> 79
CRC -> f7
